# FastAPI Patient Management API

A simple and clean **FastAPI** application for managing patient data.

## Overview

This project demonstrates a basic FastAPI setup with CRUD-like operations using patient records loaded from a JSON file.

## Features

- FastAPI with automatic Swagger UI documentation
- GET endpoints to retrieve patient data
- Simple data loading from `patients.json`
- Clean project structure
